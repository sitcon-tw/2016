Davy

Davy - 簡潔優雅的語言 —— Crystal

Crystal 是一以 LLVM 為基底的新興程式語言，語法極似 Ruby 但又同時能編譯成 Native Code，具有能夠處理自動型別推導以及靜態型別檢查的能力，同時還擁有在編譯時期的巨集展開及直接對 AST 操作的彈性，是一個想法相當前衛的語言。

Crystal 從 2012 年開始發展，現今已逐漸發展出穩定的社群，本議程將帶您認識如此美妙又高效的語言，並簡單分享講者在移植到 Windows 上時與開發者們互動的經驗談，同時介紹想要在台灣推廣 Crystal 的我們 —— Crystal-TW (http://crystal-tw.github.io)

簡介 Linux 的 Network Namespace 功能
Zuan

Zuan - 簡介 Linux 的 Network Namespace 功能

本場演講將簡介 network namespace 功能，並且簡易的示範。
不少人都知道ADSL/光纖網路其實是有多個浮動IP的，但是如果只有一台電腦，那就只能使用一個IP，如果想要同時使用多個IP，就要拿不少儲存空間跟記憶體來開虛擬機。
然而，如果使用 Linux 的 network namespace 功能，就可以只虛擬化網路的部份，用少量資源多開ADSL連線，然後將一般程式分配到這些連線下面，讓他們用不同IP上網，達到多IP上網，讓搶票，開分身，快速免空抓檔，跑網路爬蟲之類的事情事半功倍。
